# KingSpice

Authentic Indian Spices E-commerce Website.

## Features

-   **Home Page**: Hero section, featured products, and value propositions.
-   **Product Listing**: Browse our collection of premium spices.
-   **Product Details**: Detailed information about each spice.
-   **Responsive Design**: Built with Tailwind CSS for all devices.

## Tech Stack

-   Next.js 15 (App Router)
-   React 19
-   Tailwind CSS 4
-   TypeScript

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
